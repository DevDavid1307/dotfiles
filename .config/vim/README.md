# Neovim 配置

## 安装

```shell
sudo pacman -S neovim
git clone https://github.com/DevDavid1307/nvim $HOME/.config/nvim
```

## 自定义配置文件

```shell
~/.config/nvim/init.vim
```

## 插件

> vim 插件管理 https://github.com/junegunn/vim-plug

> vim 插件搜索 https://vimawesome.com/
